Generate Your YouTube Data API Key in 8 Easy Steps:

Total Time: 5 minutes

  1. Open Google Developers Console: https://console.developers.google.com/project

  2. Log into your google account, if not already or Create an Account.

  3. Create a New Project (or Choose Existing): Name it anything you want.

  4. Navigate to “APIs & Services” > “Library”

  5. Search for “YouTube Data API v3” and Click “Enable”

  6. Head to “Credentials” > “Create Credentials”

  7. Select “API key” and Click “Next”

  8. Voila! Your New API Key Appears! Copy and Save it Securely
